# my-dot-claude

Personal Claude configuration and skills.

## Skills

- `cmux-workspace-name`: helps Claude rename the current cmux workspace or tab to a ticket/task label after interviewing the user for the preferred name.

## Install

From this repository:

```bash
./scripts/install.sh
```

This syncs repository skills into `~/.claude/skills/`.

## Validate

```bash
./scripts/validate.sh
```
